Name Value Pairs:
================

	A name which maps to a unique value

	name may be defined more than once but can have one value at any given context.

	That values may be more name/value pairs.

	Address = "100 Main st."

	OBJECT:
	========

	A collection of name/value pairs.

	The simplest definition relating to JS.

	address:{
		street:'abc',
		Number:100,
		Apartment:{
			Floor:3,
			Number:301
		}
	}
